Message Generator

I wrote the JavaScript that powers the “Number Guesser” game by wiring up the UI to the game logic. When the player clicks Make a Guess, the script generates a target number, reads the player’s input, creates a random computer guess, compares which guess is closest to the target, updates and displays the scores, and shows the winner on the page. It also manages the round flow by disabling/enabling the correct buttons, resetting the input and display values for the next round, and adding plus/minus controls that keep the user’s guess within the allowed range.
